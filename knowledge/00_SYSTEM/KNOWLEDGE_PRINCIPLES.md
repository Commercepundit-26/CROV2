# Knowledge Principles

## Purpose
Defines the philosophy governing the CRO-X platform.

## Core Tenets
1. **Knowledge lives in markdown.**
2. **Code executes knowledge.** Application code should never contain hidden business logic.
3. **Agents never own business logic.** They are blind executors.
4. **Rules own business decisions.** If a CRO decision needs to be made, it must be defined as a Rule in `04_RULES`.
5. **Evidence validates.** No rule can be marked as broken without visual or DOM evidence.
6. **Recommendations explain.** The AI generates recommendations based strictly on the validated rules and evidence.
